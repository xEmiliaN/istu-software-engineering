Функциональные требования:

1) Активация охранной системы по нажатию специальной кнопки
2) Выбор службы безопасности, которая будет вызываться по кнопке
3) Вызов службы безопасности по нажатию специальной кнопки
4) Отправка уведомления от приложения о срабатывании сигнализации
5) Просмотр трансляции видео с камер видеонаблюдения
6) Сбор статистики по активации охранной системы, срабатываний сигнализации, вызовам службы безопасности
7) Генерация анализа уровня безопасности объекта, на основе собранной статистики
8) Возможность просмотра анализа и статистики

Нефункциональные требования:
1) Приложение должно быть для устройств на андроид
2) Дизайн приложения должен в стиле iOS устройств
3) Кнопка вызова службы безопасности должна иметь возможность менять службу для вызова с помощью выпадаещего списка
4) Кнопка связи с техническим консультантом должна быть в самом низу, специально выделяться серым цветом 
5) На экране просмотра видео с камер, должны быть две функциональные кнопки
