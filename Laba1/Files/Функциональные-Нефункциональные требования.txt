Функциональные требования:

1) Активация охранной системы по нажатию специальной кнопки (!)
2) Выбор службы безопасности, которая будет вызываться по кнопке
3) Вызов службы безопасности по нажатию специальной кнопки (!) Возможность вызова службы безопасности по нажатию кнопки "ВЫЗОВ"
4) Отправка уведомления от приложения о срабатывании сигнализации
5) Просмотр трансляции видео с камер видеонаблюдения
6) Сбор статистики по активации охранной системы, срабатываний сигнализации, вызовам службы безопасности (!) Сбор статистики по активации (!) Сбор статистики срабатываний сигнализации (!) Сбор статистики по вызовам службы безопасности
7) Генерация анализа уровня безопасности объекта(Уточнить), на основе собранной статистики и её просмотр

Нефункциональные требования:

+ Анализ должен выводиться в формате 
1) Приложение должно быть для устройств на андроид (не ниже версии 10)
2) Дизайн приложения должен в стиле iOS устройств ((См. дизайн-макет))
3) Кнопка вызова службы безопасности должна иметь возможность менять службу для вызова с помощью выпадаещего списка
//При длительном нажатии на кнопку должен 

4) Кнопка связи с техническим консультантом должна быть в самом низу, специально(кыш) выделяться серым цветом 
5) На экране просмотра видео с камер, должны быть две функциональные кнопки ВЫЗОВ и Вторая которая
